A basic link sends to you to another page; the code is 'inline', or within the line of text. The anatomy of a link:

<a href="anotherPage.html">the text that will be a link</a>

Links to internal pages, in the same folder, can be listed as simply the filename.

To link into a subfolder (perhaps of images), use the nameOfFolder/nameOfFile.extension

Links to external pages (on another site) should always include the full URL including the http:// part.
